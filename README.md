# Instagram Reach and Engagement Analytics Dashboard

## Overview

An interactive Power BI dashboard developed to analyze Instagram post performance, audience engagement, reach, impressions, and content effectiveness. The dashboard transforms social media data into actionable insights, helping users understand audience behavior and optimize content strategies.

## Project Objective

To build a social media analytics solution that enables the analysis of engagement patterns, content performance, and audience reach, supporting data-driven decision-making for social media growth.

## Dataset

The dataset contains information related to:

* Post Captions
* Likes
* Comments
* Shares
* Saves
* Reach
* Impressions
* Engagement Rate
* Reach Sources

## Key Insights

* Reach and impressions varied significantly across different content types.
* Engagement rate helped identify high-performing posts and audience preferences.
* Likes represented the largest share of overall audience engagement.
* Reach source analysis highlighted the most effective channels driving visibility and audience growth.
